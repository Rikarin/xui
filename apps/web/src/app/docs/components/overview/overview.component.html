What is xUI?

xUI is a customizable Angular 18 UI library with a gaming look, which can be easily adapted to your brand. It contains a default dark and light theme, as well as SCSS functions to generate your own theme in a few steps. Additionally, the theme can be switched at runtime.

What you will get?
  • Light and Dark default themes easily customizable
  • 33+ Angular UI Components ready to be used
  • Great Theming Engine - creating new components just in few steps
  • Configurable Options - tweaking the components in various ways
  • Material Design Icons - ready to be used MDI Search
