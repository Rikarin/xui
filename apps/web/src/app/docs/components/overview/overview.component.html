Get started by reading Getting Started

What is xUI?

xUI is a customizable Angular 16 UI Library with gaming look which can be easily adapted to your brand. Contains default dark and light theme and also SCSS functions to generate your own theme in few steps. Which can be also switch on runtime.

What you will get?
  • Light and Dark default themes easily customizable
  • 33+ Angular UI Components ready to be used
  • Great Theming Engine - creating new components just in few steps
  • Configurable Options - tweaking the components in various ways
  • Material Design Icons - ready to be used MDI Search
